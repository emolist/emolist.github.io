About

A Computer Science Senior Design Project

Emolyst is a CRM (Customer Relationship Management) application for legal entities and it provides analysis of their customers based on their facial expression and age-gender categories. Emolyst uses a mixture of recent technologies that are accepted by the open-source community to extract facial expression units and to detect age and gender information. Emolyst gathers the data from live camera streams connected to client hardware which is a single-board computer and processes it on the single-board computer. Then processed data is transferred to Emolyst server to create relations between items and user groups. Users of the Emolyst will be able to obtain statistics and graph analysis which are prepared by Emolyst.
